# 🚀 BeamMCP

BeamMCP is a lightweight MCP (Model Context Protocol) server that connects AI agents to cloud execution tools.

It allows AI systems to:
- Run tools via MCP protocol
- Process requests like domain search
- Extend with custom tools easily

---

## ⚙️ Features

- MCP-compatible API (`/mcp`)
- Tool system (extensible)
- Example tool: domain availability check
- Built with Express.js
- Ready for cloud deployment (Vercel / Render / VPS)
